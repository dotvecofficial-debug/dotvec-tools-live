DOTVEC TOOLS – FINAL HOMEPAGE AND PAGE MANAGER BUILD
====================================================

Run on Windows:
1. Extract the ZIP fully.
2. Open the extracted folder.
3. Run START_DOTVEC.bat.
4. Keep the terminal open.
5. Open the localhost URL displayed in the terminal.

Admin:
/admin/login
/admin/home       Full homepage editor
/admin/pages      Add, publish, hide and delete static pages
/admin/settings   Theme and flat card colors
/admin/seo        Overall SEO manager
/admin/sitemap    Sitemap, robots.txt and .htaccess
/admin/tools      Tool content and status controls
/admin/analytics  First-party analytics

The homepage editor now controls all homepage text, cards, lists, section visibility and section order.
